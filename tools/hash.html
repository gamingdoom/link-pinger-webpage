You can use this: Compute File Hash (FileReader) to compute the MD5/CRC32/SHA256. for a given file. You can also use this: Online String MD5 Hash to compute the MD5 hash for strings.

     $s = $_GET['s'];

  }
  $data = array();
  foreach (hash_algos() as $v) { 
          $r = hash($v, $s, false);
